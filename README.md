# Gilly's Cafe Website

A modern, responsive café website inspired by Café Calibre, featuring a beautiful design with all the essential sections for a restaurant/café business.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, elegant design with smooth animations
- **Menu Filtering**: Interactive menu with category filters (Breakfast, Lunch, Dinner, Drinks, Coffee)
- **Multiple Sections**:
  - Hero section with call-to-action buttons
  - About/Story section
  - Restaurant locations with opening hours
  - Best sellers showcase
  - Full menu with filtering
  - Customer testimonials
  - Contact information
  - Footer with social media links

## Getting Started

1. Simply open `index.html` in your web browser
2. No build process or dependencies required - it's a pure HTML/CSS/JavaScript website

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #8B4513;    /* Main brand color */
    --secondary-color: #D4A574;  /* Accent color */
    --accent-color: #F5E6D3;     /* Light background */
}
```

### Content
- Update text content directly in `index.html`
- Add/remove menu items in the menu section
- Modify contact information in the contact section
- Update opening hours in the locations section

### Images
Replace placeholder emojis with actual images by:
1. Adding image files to an `images` folder
2. Updating the `menu-image` divs to use `<img>` tags instead of placeholder divs
