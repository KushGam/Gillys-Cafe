# Deploy to Vercel

## Quick Deploy via Dashboard (Recommended)

1. Go to https://vercel.com/new
2. Sign in with GitHub
3. Click "Import Git Repository"
4. Select `KushGam/Gillys-Cafe`
5. Click "Import"
6. Vercel will auto-detect it's a static site
7. Click "Deploy"
8. Your site will be live in ~2 minutes!

## Deploy via CLI

1. Run: `vercel login`
2. Choose "Continue with GitHub"
3. Complete browser authentication
4. Run: `vercel --prod`
